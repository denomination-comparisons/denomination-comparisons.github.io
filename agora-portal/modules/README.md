# Agora Modules

This folder contains specification documents and shared resources used across
the Agora Portal front-end and back-end.

Contents:

- `tradition-vectors.md`: How to encode denominational and philosophical background.
- `interpretation-layer.md`: How to build structured theology JSON.
- `theology-schema.json`: Schema for `/core/theology/*.json`.
- `safety-schema.json`: Schema for crisis-resource JSON files.
